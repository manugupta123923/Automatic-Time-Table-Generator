# Automatic Time Table Generator

A Python-based application that automatically generates class timetables
while avoiding subject and faculty conflicts.

## Features
- Automatically generates academic timetables
- Avoids subject and faculty clashes
- Simple and efficient scheduling logic
- Easy to use

## Technologies Used
- Python

1. Clone the repository
2. Open the project folder
